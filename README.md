# 🛒 Smart Trolley System using ESP32 & RFID

## 📌 Description

This project is an IoT-based smart shopping system that automates billing using RFID technology.

## 🚀 Features

* 📡 RFID product detection
* 💡 LCD display for product info
* ➕ Add/remove items using card
* 🌐 Web-based billing system

## 🛠️ Tech Stack

* ESP32
* Arduino IDE
* Node.js
* MongoDB

## ⚙️ How It Works

1. User taps RFID card
2. Product is added/removed
3. LCD shows details
4. Bill is generated automatically
